# Diseño de intercambiador de calor
En el presente trabajo, se desarrolla el diseño de un intercambiador de calor orientado a recuperar la energía térmica de los gases de escape a 450◦C de un grupo electrógeno a GLP, con el objetivo de calentar agua de red para almacenamiento desde 15◦C hasta 70◦C. Para ello se aplicaron dos métodos siendo estos: el método de temperatura media logarítmica y método de efectividad NTU

## Resultados
 * Relación aire-combustible: 15.19
 * Potencia termica de intercambiador: 8kW
 * Area de transferencia de calor: 2.29m^2
 * Diametro nominal de tubos: 1/4 in
 * Numero de tubos en banco de tubos: 37
 * Longitud de tubos: 1.42 m
 * Material de tubos: ASTM A312 (k=14.2 W/mK)
 * Flujo de gases calientes: 4.1e-04 m^3/s
 * Flujo de agua caliente: 3.51e-05 m^3/s
 * Potencia de bomba de agua: 6.33e-08 W
 * Caida de presión de gases residuales: 94.77 Pa
